# My Coding Practice

This folder is a collection of small projects and exercises I’ve worked on to sharpen my skills. 

### 🚫 No AI
I decided to keep this specific folder **100% human-written**. 

I didn't use ChatGPT, Copilot, or any other LLMs to write these scripts. Everything you see here was built using only my knowledge, documentation, and a lot of trial and error. 

### Why I did this:
* **To struggle a bit:** I wanted to make sure I actually understand the logic, not just how to prompt for it.
* **To learn the "why":** Debugging things myself helps me remember the solutions much better.
* **To keep it raw:** They are absolutly not "perfect" or "optimized by an algorithm," but they are mine.

### What's inside?
Just a bunch of small experiments, logic puzzles, and mini-apps. Each folder is a step in my learning journey.
