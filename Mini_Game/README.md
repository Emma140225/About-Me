# Warrior vs. Fairy: My First C# Game

This is a rudimentary battle simulator and represents my very first steps into programming with C#. 

### 🕹️ About the Project
In this game, a **Warrior** and a **Fairy** face off in a turn-based battle. I built this to practice the core fundamentals of C#, focusing on logic flow and how different objects interact with each other.

**A note on AI usage:**
To keep the learning experience authentic, I didn't use AI for the game's logic or structure. I only used AI support for the `MessageBox` method inside the `Helper` class, which creates a visual frame (border) around the text in the console to make it look more organized.

### 🧠 Features & Learning Goals
* **Classes & Objects:** Defining the Warrior and the Fairy with their own attributes.
* **Game Loop:** Managing the battle until one character's HP reaches zero.
* **Console Styling:** Using a helper class to format console output with frames for better readability.
* **Randomization:** Implementing `Random` to determine damage and turn outcomes.
